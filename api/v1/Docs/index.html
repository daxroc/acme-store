User

Entry Point Description URI METHOD Parameters Pre-Conditions Post-Conditions
/api/v1/user Returns list of all users user GET None Login and requires administration privileges. None
/api/v1/user/view Returns user details user/view GET None Login None
/api/v1/user/create Creates user record user/create PUT None Login None
/api/v1/user/update Creates user record user/update POST None Login, post body contains updated version json object to be updated. None
user/delete/<uuid> Deletes user record user/delete DELETE Login, administration privileges. No protection for users who delete their own account. Record identified with <uuid> will be permanantly deleted.
/api/v1/user/urls Returns login/logout urls for current user user/urls GET None Login

Product

Entry Point Description URI METHOD Parameters Pre-Conditions Post-Conditions
/api/v1/product Returns list of all objects product GET None None None
/api/v1/products/view Returns single product details products/view GET None None None
/api/v1/products/create Creates product record products/create POST

Body contains json object of new record.

  {
    "product_id": 2,
    "name": "Dynamite",
    "manufacturer_id": 1,
    "price": 5,
    "qty": 50000,
  }
Login, Administration privilege
/api/v1/products/update/<uuid> Updates product record products/update PUT

Body contains json object of product update.

Login, post body contains updated version json object to be updated. None
/api/v1/products/delete/<uuid> Deletes products record products/delete DELETE None Login, administration privileges. Record identified with <uuid> will be permanantly deleted.

Manufacturer

Login, Administration privilege

post body contains updated version json object to be updated.

Login None
/api/v1/manufacturer/delete/<uuid> Deletes products record manufacturer/delete DELETE None Login, administration privileges. Record identified with <uuid> will be permanantly deleted.

Orders

Entry Point Description URI METHOD Parameters Pre-Conditions Post-Conditions
/api/v1/orders Returns list of all order objects for user orders GET None Login None
/api/v1/orders/view/<uuid> Returns orders object for object identified by <uuid> orders/view GET <uuid> required Login None
/api/v1/orders/list Returns administration list of orders objects for all users orders/list GET None Login, Administration None
/api/v1/orders/create Creates single record orders/create POST

Body contains json object of new record.

{
  "updated": "04-08-2015 00:45",
  "cart":
  {
      "cart_total": 500,
      "cart_items":
      [
          {
              "manufacturer_id": 1,
              "price": 5,
              "name": "Dynamite",
              "qty": 100,
              "product_id": 2
          }
      ]
  },
  "created": "04-08-2015 00:45",
  "user_id": "185804764220139124118"
}
Login
/api/v1/orders/update/<uuid> Updates orders record identified by <uuid> orders/update PUT

post body contains updated version json object to be updated.

Login None
/api/v1/orders/delete/<uuid> Deletes orders record identified by <uuid> orders/delete DELETE <uuid> required Login Record will be permanantly deleted.
